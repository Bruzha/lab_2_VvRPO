Лабораторная работа 
Использование системы git, Организация удалённого репозитория
Задания:
Зарегистрируйтесь на сервисе github. 
Создать пустой удалённый репозиторий. 
Создайте проект на основе предоставленного преподавателем исходного кода.
Инициализируйте локальный репозиторий.
Создайте несколько коммитов.
Синхронизируйте локальную ветку с основной веткой в созданным удалённым репозитории.
Создать новую ветвь в локальном репозитории.
Создать несколько коммитов в новой ветви.
Вытолкните новую ветвь в удалённый репозиторий
Объедините новую ветку с основной веткой на удалённом репозитории.
Получите изменения из основной ветви удалённого репозитория в локальный.
На сайте сервиса:
Присоедините к работе над удалённым репозиторием коллегу 
Создать новую ветвь и передать её для работы коллеге
Коллега должен внести изменения в свою ветку и подать pull request.
Принять pull request, выполнив слияние ветки с изменениями, которые внёс коллега, с основной веткой
